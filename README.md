# PruebaIngresoNewShore
Para para ejecutar la aplicación se debe crear la base de datos ejecutando los siguientes Scripts en una Base de datos SQL Server:
1. PruebaIngresoNewShore\ScriptsSQL\CreateDataDase.sql
2. PruebaIngresoNewShore\ScriptsSQL\InsertDataRoute.sql

luego se debe cambiar el valor la cadena de conexión a la base de datos, modificando el valor de la variable "DefaultConnection" que se encuentra el archivo appsettings.json
por último se publica el sitio y se despliega en el IIS o se depura en VisualStudio
